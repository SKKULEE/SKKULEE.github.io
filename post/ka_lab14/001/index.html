커널 아카데미 AI Lab #001 3–9일차 파이썬 기초와 데이터 시각화, 웹 크롤링

2025.06.09.(MON)

Python 기초

 Python과 Anaconda를 설치하는 것으로 본격적인 교육이 시작됐다. 비전공자와 전공자가 섞여있는 과정인만큼 첫 날(3일차)은 아주 기초적인 부분부터 다뤘으며, Python의 자료형, 기초 문법, 반복문 및 함수 선언 등을 교육했다. 전공자 입장에서는 너무 기초적인 내용이라 딱히 새로운 것은 없었다.

Pandas 기초

 4일차에 들어서는 기초적인 AI 개념과 Pandas 모듈 기초에 대해 다뤘다. Pandas의 기초 데이터 포맷인 DataFrame에 다루고, 데이터를 시각화하는 여러 방법을 소개, 그리고 적합합 시각화 형태를 찾을 수 있는 웹 사이트 data-to-viz.com도 소개받았다.

 이후 TML(Traditional Machine Learning), Deep Learning의 개념과 structured data, unstructured data의 차이, 기계학습에서 데이터 수집의 중요성, 오픈 API를 이용한 공공 데이터 수집 기초 등을 다뤘으며, Awesome Public Datasets, Google AI Datasets, Google Dataset Search, Kaggle Competition Datasets과 같은 data collection 추천 또한 받을 수 있었다.

데이터 수집 기초

 5일차는 오전, 오후 모두 자율학습을 진행해서 따로 교육 내용은 없었고, 6일차에는 데이터 수집 기초 교육에 들어갔다. 이전에 이야기했던 데이터의 중요성에 대해 다시 한 번 강조했으며, 데이터를 수집하고 변형하여 양질의 데이터를 얻어내는 과정에 대해 설명했다. 그 과정에서 남들도 다 사용해본 공개된 데이터셋으로 남들도 다 만들어본 모델을 만드는 것은 딱히 경쟁력이 없으며, 데이터 수집부터 고민한 흔적이 있는 프로젝트가 포트폴리오에 더 적합하고 경쟁력 있다는 조언 또한 구할 수 있었다.

 이후로는 (웹 프로젝트 진행 목적이 아니라) 데이터 크롤링을 위한 웹 구조 및 구성에 대한 간략한 설명이 이어졌고, 웹 크롤링 실습을 위해 공개되어있는 웹 사이트인 toscrape.com에 대한 소개가 있었다.

데이터 수집 실습

 7–9일차는 Python의 BeautifulSoup 모듈을 사용해 toscrape.com, webscraper.io에서 웹 데이터를 수집하는 실습을 진행했다. Jupyter Notebook을 활용한 단계적이고 상세한 교육이 진행된 점이 좋았다. 진행 중 원하는 데이터(상품명, 가격 등)를 특정해서 읽는 방법, 빈 데이터를 처리하는 방법(평균으로 처리하기, 0으로 처리하기 등), 특정한 형태로 변환하는 방법 등을 배울 수 있었다.

후기

 실전적인 교육내용이 난잡하기 않고 체계적으로 진행되어서 만족도가 매우 높았다. 준비가 아주 잘 된 느낌이 났고 강의력도 뛰어났다. 그런만큼 기억에 많이 남았는데, 잊지 않도록 실습자료를 활용해 다시 한 번 복습을 진행할 필요를 느꼈다.
