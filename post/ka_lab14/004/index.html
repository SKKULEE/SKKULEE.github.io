커널 아카데미 AI Lab #003 21–23일차 Git

2025.06.27.(FRI)

Git

 21일차는 자율학습이 진행되었고 22일차와 23일차는 최우영 강사님의 실시간 Git 실습 강의가 진행되었다. 그 중 첫 날은 Git 소개 및 설치, Github 레포지토리 연결, commit과 push 대한 기초적인 강의가 진행되었고 팀 개발 효율성을 끌어올리기 위한 conventional commit에 대한 소개와 실습이 그 뒤를 따랐다. 둘째 날은 local 혹은 remote 저장소에서 conflict가 일어날 경우 대처 방법, github organization으로 팀을 이루고 issue format, conventional commit, pull request 등을 통해 통일된 형식으로 체계적인 협업을 하는 방법을 배울 수 있었다.

후기

 기술적으로는 수업에서 다뤘던 git의 기능과 명령어들을 이미 거의 알고 있던지라 특별히 내용정리 하거나 요약할만한 것은 없었다. 하지만 체계적인 conventional commit과 issue, pull request를 통한 코드 리뷰 등 외적인 부분에서 보다 효율적으로 협업하는 방법에 대해 알아갈 수 있었고 이에 크게 만족스러운 시간이였다.
